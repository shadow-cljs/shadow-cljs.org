### Usage

Develop:

```bash
yarn
yarn watch # HTTP server listening on 7000

# generate HTML from another terminal after watch server started
yarn page
# add soft link for the CSS file
(cd target && ln -s ../entry/)
```

Build:

```bash
yarn build-local
yarn http-server dist
```
